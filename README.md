# Laravel REST API

This project is a RESTful API developed using Laravel.

## 🔹 Features
- User authentication using Laravel Sanctum
- CRUD operations (Create, Read, Update, Delete)
- REST API structure
- Database relationships handling
- Eager Loading implemented to solve N+1 query problem

## 🔹 Tech Stack
- Laravel
- PHP
- MySQL
- Postman (API testing)

## 🔹 Key Highlights
- Optimized database queries using Eager Loading
- Clean and structured API responses
- Followed MVC architecture

## 🔹 Purpose
This project demonstrates my backend development skills, API design, and database optimization techniques.
